## Question Types

### Multiple Choice
- Question must reference the image directly (e.g., "Based on the diagram...", "Looking at the image...")
- Options should be clearly distinct
- Include plausible distractors based on common misconceptions
- Reference specific visual elements

### Short Answer
- Require students to interpret or describe visual information
- Expected answer: 1-3 sentences
- Focus on observation, interpretation, or analysis

### True/False
- Statement about image content that is unambiguously true or false
- Reference specific visual details
- Include brief explanation

### Essay/Extended Response
- Require analysis, comparison, or synthesis of visual information
- Include grading rubric criteria
- Specify expected length

## Formatting Guidelines

- Use clear references to image elements ("the diagram shows...", "according to the chart...")
- For mathematical/technical images, use LaTeX formatting: $...$ for inline, $$...$$ for display
- Include context clues that help students locate relevant parts of the image
- Ensure questions cannot be answered without viewing the image
